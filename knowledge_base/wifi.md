# Wi-Fi Troubleshooting

If a laptop cannot connect to Wi-Fi:

1. Restart the laptop.
2. Check if Wi-Fi is enabled.
3. Verify Airplane mode is disabled.
4. Forget and reconnect to the network.
5. Restart the router.
6. Update network drivers.
7. Run Windows Network Troubleshooter.

If the problem still exists after these steps, create an IT support ticket.
